推荐使用[nginx](http://nginx.org/)反向代理  

Win hosts配置项：
```
127.0.0.1   pc.test.com
```

nginx.conf配置项：  
```
server {
    listen  80;
    server_name  pc.test.com;
    location / {
        proxy_pass   http://127.0.0.1:8002;
    }
}
```

## Build Setup

``` 
# 安装依赖项
npm install --registry=https://registry.npm.taobao.org

# 开发模式 localhost:8002
npm run dev

# 推荐使用forever
npm run start

# 打包项目
npm run build
```
